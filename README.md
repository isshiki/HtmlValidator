HtmlValidator
====================================

HTMLソース（基本的にHTML5）のタグ階層やタグの記載にミスがないか基本的な検証を行う機能を提供する。

目的
-------------------------------------------------------------------

HTMLデータをDOMデータに変換してLINQなどで扱う際に、元々のHTMLタグの書き損じや階層構造の破たんの問題があると、いろいろと問題がある。

そのような、基本的な問題をシンプルに検証して、HTMLページの形式でデフォルトブラウザーにより通知すること。


ソースフォルダーの内容
-------------------------------------------------------------------

- Tester： この機能を検証するためのWindowsネイティブアプリ（WPF）
- ルートフォルダー： Testerフォルダー以外のファイルが、この機能を実装しているソースファイル群
